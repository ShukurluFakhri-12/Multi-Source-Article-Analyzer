# Multi-Source Article Analyzer

A Python-based automation tool that:

- Scrapes article content from multiple news sources
- Extracts title and full text
- Calculates word count and estimated reading time
- Identifies top keywords
- Exports structured report to CSV

---

## Features

- Web scraping using `requests` and `BeautifulSoup`
- Basic text cleaning and keyword frequency analysis
- Reading time estimation (200 words/minute)
- CSV report generation using `pandas`

---


---

##  Technical Highlights

- HTTP request handling with timeout control
- HTML parsing and structured data extraction
- Basic NLP-style keyword frequency analysis
- Automated reporting pipeline
